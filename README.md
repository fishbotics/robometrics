# Robometrics

Robometrics 
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install robometrics.

```bash
pip install robometrics
```

## Usage

```python
import robometrics

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
